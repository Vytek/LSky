Basic and free dynamic sky system.

# Features:
- Day and night cycle. 
- Curve and gradient system. 
- PBS Atmosphere. 
- Night Color. 
- 2d Moon. 
- Outer Space 
- LDR & HDR Support. 
- Linear & Gamma Color Space Support.
